`tmux` es un multiplexor de terminales, basado en múltiples sesiones de Terminal en una única ventana. Desde una única ventana de iTerm2.app o Terminal.app puedes tener múltiples sesiones activas, cambiar de una a otra, verlas simultáneamente, entrar en una y desconectarte (se mantienen ejecutando en segundo plano) y volver a conectarte a ella desde otra ventana de iTerm2/Terminal diferente.

## Instalación

Instalar tmux a través de HomeBrew de OSX

```zsh
% brew install tmux
% tmux -V
```

Abrir una nueva sesión con `new`. Puedes salir de ella con `exit`.

```zsh
% tmux new [-s nombre_de_sesion]
```

Dárle órdenes (comandos):

```cli
<CTRL+B> + Comando
```

### Sesiones

|Comando| Descripción|
|-|-|
| tmux [new] | Nueva sesión |
| tmux new -s [nombre] | Ponerle nombre a la sesión |
| tmux attach # | Iniciar otra sesión en paralelo |
| tmux attach -t [nombre] | Conectar con una sesión |
| tmux ls | Listado de sesiones |
| exit | salir de la sesión |

### Manejo de ventanas

|Comando| Descripción|
|-|-|
| CTRL+B d | Detach |
| CTRL+B c | Nueva ventana |
| CTRL+B n | Siguiente ventana |
| CTRL+B w | Listar ventanas |
| CTRL+B , | Cambiar el nombre de una ventana |

### Manejo de paneles

|Comando| Descripción|
|-|-|
| CTRL+B % | Dividir paneles verticalmente (izda/dcha) |
| CTRL+B " | Dividir paneles horizontalmente (arriba/abajo) |
| CTRL+B x | Ocultar panel |
| CTRL+B q | Mostrar números de paneles |
| CTRL+B Flecha | Cambiar entre paneles |
